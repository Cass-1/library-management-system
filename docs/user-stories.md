# User Stories

## Registration and Verification

**As a member** I want to be able to sign up for an account so I can access the library services.

**As a staff member** I want to be able to register as a staff member so I can manage charges/discharges and the book catalogue.

**As a library manager** I want to be able to register as a manager so that I can manage staff, access staff services, and view analytics.

**As an admin** I want to be able to register as an admin so that I can manage managers.

## Charge and Discharge Books

**As a member** I want to be able to checkout and return books so that I can access the library's catalogue.

## Reservations

**As a member** I want to be able to reserve books to that they will be auto-checked out when the book is next available.

**As a member** I want to be able to cancel a book reservation that I have previously made.

**As a staff member** I want to be able to cancel member reservations so that books can be recalled.

## Recalls

**As a staff member** I want to be able to issue recalls so that books can be auto-returned from members and clear all pending reservations.

## Renewals

**As a member** I want to be able to request a renewal so that I can continue to check out a book past its return date.

**As a staff member** I want to be able to cancel renewal requests in order that the book can be returned on time.

## Inter-Library Loans

**As a staff member** I want to be able to request inter-library loans so that the library can temporarily borrow books it doesn't have.

## Fines

**As a member** I want to be able to pay fines for overdue books so that I can check out new books.

**As an admin** I want to be able to control the fine rates so that I can reduce or increase the cost of fines.

## Book Requests

**As a member** I want to be able to request a book that is not in the library's catalogue so that it may be added.

**As a library manager** I want to be able to see the most often requested books so that I can determine wether it is worth it to add a book to the catalogue.

## Analytics and Statistics

**As a library manager** I want to be able to see membership trends and book trends to be able to report on membership and see what types of books I should add to the catalogue.

## Manage Memberships

**As a library manager** I want to be able to add/remove staff members so that I can control who has access to staff services.

**As an admin** I want to be able to add/remove library managers so that I can control who has access to library manager services.

## Search and Filter Inventory

**As a member** I want to be able to search the catalogue so that I can find books to check out.

**As a staff member** I want to be able to search the catalogue, so that I can keep tabs on inventory.

**As a library manager** I want to be able to search the catalogue, so that I can add/remove books.

## Read EBooks

**As a member** I want to be able to read EBooks that I have checked out.

**As a staff member** I want to be able to read EBooks so that I can ensure their quality.

## Add/Remove Books

**As a library manager** I want to be able to add/remove books, so that I can update the catalogue.
